# W — Fashion Store

Toko online Brand W berbasis Next.js + Supabase.

## Tech Stack
- Next.js 14 (TypeScript)
- Supabase (Auth + Database)
- Tailwind CSS
- Midtrans (Payment)

## Setup

1. Clone repo
   git clone https://github.com/cinaganteng22/wiliiiii
   cd wiliiiii

2. Install dependencies
   npm install

3. Buat file .env.local
   NEXT_PUBLIC_SUPABASE_URL=isi_url_kamu
   NEXT_PUBLIC_SUPABASE_ANON_KEY=isi_key_kamu

4. Jalankan
   npm run dev
